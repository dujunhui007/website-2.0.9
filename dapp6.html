Introduction:

SubSpace (SPC) is an intelligent contract platform that focus on block chain games. It aims to create an independent and decentralized ecosystem around intelligent contracts of game information, game community, gaming Peripherals, game on the chain, game competition, etc. It is full autonomy for all the people, verifiable for all the trades, unforgeable for all the data, irreversible results for all game agreement.
